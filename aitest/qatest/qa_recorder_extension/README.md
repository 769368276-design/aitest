# QA Recorder 扩展（MVP）

## 安装（Chrome）

1. 打开 `chrome://extensions/`
2. 打开右上角“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择本仓库目录下的 `qa_recorder_extension/`

## 使用

1. 打开要录制的系统页面
2. 点击浏览器右上角扩展图标，点“开始录制”
3. 正常操作页面（点击、输入、下拉、回车等）
4. 点“停止”，复制 JSON
5. 在平台打开“录制导入”页面，粘贴 JSON 生成用例步骤

## 说明

- 这是线性步骤录制器（第一期），后续可以逐步扩展为积木式/可视化编辑。
